## 📌 공통 적용 사항 (모든 페이지)
- 사이트 전체 테마 디자인 시스템 구축  
  - CSS 변수(:root)를 사용하여 컬러 팔레트 및 폰트 설정을 중앙 관리
  - 어두운 배경 기반의 SAW 콘셉트 UI 일관성 유지
  - 주요 컬러:  
    - Background: #000  
    - Menu: #293627  
    - Hover / Accent: #8B0000  

- Google Fonts 적용  
  - Noto Sans KR 폰트 로드
  - 모든 페이지에서 동일한 폰트 사용으로 가독성 및 시각적 통일성 확보

- 공통 네비게이션 바 구조 통일  
  - Home / Login / Instagram Login / Post / Join 메뉴 구성 유지
  - 메뉴 hover 효과 및 구분선(border) 스타일을 테마 색상으로 통일
  - 페이지 이동 시 사용자 혼란 최소화

- 고정 레이아웃 기반 중앙 정렬 구조 유지  
  - width: 1200px 기준의 중앙 정렬 컨테이너 사용
  - 모든 페이지에서 동일한 레이아웃 기준점 유지

- 로고 링크 공통 적용  
  - 좌측 상단 고정 로고 배치
  - 클릭 시 홈(home.html)으로 이동하도록 구성

- 입력 폼 UI 스타일 통일  
  - 입력 필드 배경을 어두운 톤으로 통일
  - 포커스 시 테두리 색상 변화로 입력 상태 명확화
  - hover / focus 효과를 통한 사용자 인터랙션 강화

- 공통 푸터 구성  
  - 저작권 문구 및 문의 정보 표시
  - 교육용 프로젝트임을 명시
  - 모든 페이지 하단에 동일한 푸터 적용으로 완성도 향상

- 프론트엔드 중심 과제 구조  
  - 모든 페이지는 UI/UX 구현에 초점을 둔 프론트엔드 결과물
  - 실제 인증, 데이터 저장, 파일 업로드, 서버 연동 기능은 포함되지 않음

### home.html (홈 페이지)
- meta description 및 Open Graph(og:title/description/image/type) 추가로 공유/미리보기 정보 보강
- 고정 폭(width:1200px) → 반응형(container max-width + width:95%)으로 개선
- 히어로 섹션(메인 이미지 + 중앙 텍스트) 추가하여 첫 화면 메시지 강화
- 캐러셀 이미지가 찌그러지지 않도록 object-fit: contain + 중앙 정렬로 표시 방식 개선
- 경고 문구(테마 안내) 영역 추가

### join.html (회원가입 페이지)
- 회원가입 폼 UI 시각적 개선  
  - 입력 필드 하단 테두리 스타일 유지하면서 테마 색상 적용
  - 버튼 영역 시각적 구분 강화
    - 회원가입 버튼: 강조 색상(붉은색 계열) 적용
    - 가입취소 버튼: 어두운 배경 + 테두리 처리로 대비 확보
  - hover 시 색상 변화 추가로 사용자 인터랙션 강화
 
  ※ 회원가입 데이터 처리 및 서버 연동 기능은 포함되어 있지 않습니다.
